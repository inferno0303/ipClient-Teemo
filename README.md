# ipClient-Teemo-ClientApp
ipClient-Teemo，轻量化的桂电出校器客户端，仅保留预拨号功能，基于python+qt5重写，纯属为了好玩的。右侧👉 release 可以下载。

## 功能

输入任意 MAC 地址，或者选择本机网卡 MAC 地址，点击按钮，即可完成预拨号。适合没有路由器的同学用。
